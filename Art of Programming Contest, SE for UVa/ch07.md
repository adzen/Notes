* Big Mod
  
(A * B * C) mod N == ((A mod N) * (B mod N) * (C mod N)) mod N

  * 通常用於費馬小定理 (Fermat Little Test)
  
  R = B^P mod M (B^P 是個很大的數字)
  
    * 使用分而治之法可以很快地計算出 R
    
long bigmod(long b, long p, long m) {
  if (p == 0)
    return 1;
  else if (p%2 == 0)
    return square(bigmod(b, p/2, m)) % m;  // square(x) = x * x
  else
    return ((b % m) * bigmod(b, p-1, m)) % m;
}
